# research

### Navigating the Research repo
```bash
research/
│
├── 00_roadmap/                     # Your personal research map
│   ├── learning-path.md            # Step-by-step goals (beginner → advanced)
│   ├── milestones.md               # Key checkpoints in your research journey
│   └── journal/                    # Daily/weekly logs of what you're learning
│
├── 01_foundations/                 # Core knowledge you must master
│   ├── computing_basics.md         # OS, networking, processes, memory, etc.
│   ├── programming/                # Notes by language (C, Python, ASM, etc.)
│   ├── math/                       # Discrete math, algebra, calculus, crypto math
│   └── security_basics.md          # CIA triad, threat models, security mindset
│
├── 02_systems/                     # Deeper technical areas
│   ├── operating_systems/          # Windows, Linux, macOS internals
│   │   ├── windows/
│   │   ├── linux/
│   │   └── macos/
│   ├── networking/                 # Protocols, packet structure, attacks
│   ├── compilers/                  # Binary → Assembly → Machine code
│   └── memory_management/          # Stacks, heaps, buffers, allocators
│
├── 03_exploit_dev/                 # Where vulnerability research gets hands-on
│   ├── buffer_overflows/           # Classic stack smashing notes, examples
│   ├── heap_exploitation/          # glibc, ptmalloc, unlink, tcache, etc.
│   ├── format_string/              # printf, arbitrary write, exploitation
│   ├── use_after_free/             # UAF, double free, practical cases
│   ├── shellcode/                  # Writing, encoding, polymorphic shellcode
│   └── modern_defenses/            # DEP, ASLR, CFI, mitigations & bypasses
│
├── 04_malware_analysis/            # Learning attacker tools & tradecraft
│   ├── static_analysis.md
│   ├── dynamic_analysis.md
│   ├── reverse_engineering/        # IDA, Ghidra, Radare2 notes
│   └── case_studies/               # Famous malware (Stuxnet, WannaCry, etc.)
│
├── 05_ai_and_security/             # Spectra’s unique angle
│   ├── adversarial_ml.md           # How attackers manipulate AI
│   ├── ai_for_exploitation.md      # Using AI/ML to find vulns
│   └── anomaly_detection.md        # AI for defense (IDS/IPS, malware detection)
│
├── 06_field_notes/                 # Active “working notes”
│   ├── exploits_read/              # Papers & writeups you've studied
│   ├── conference_notes/           # Black Hat, Defcon, CCC notes
│   ├── experiments/                # Code snippets, PoCs, failed attempts
│   └── personal_reflections.md     # Thoughts, struggles, breakthroughs
│
├── 07_references/                  # The big curated library
│   ├── papers/                     # PDFs & summaries of academic papers
│   ├── books/                      # Key books w/ chapter notes
│   ├── blogs/                      # Summaries of useful blog posts
│   ├── cheat_sheets/               # One-pagers for fast recall
│   └── glossary.md                 # Your own living dictionary
│
└── README.md                       # Overview of the repo & how to use it
```
